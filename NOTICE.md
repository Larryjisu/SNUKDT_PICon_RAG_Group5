# Notice about the upstream framework

This repository does **not** bundle the full PiCon framework source code.

It only contains the integration layer and project-specific code written for:
- a Jiyu Kim persona server
- a wrapper that launches PiCon evaluation against that server

To run the full evaluation pipeline, prepare the upstream PiCon framework separately.
Please keep the original repository attribution, license terms, and authorship with the upstream project.
